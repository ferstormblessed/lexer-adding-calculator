# lexer-adding-calculator

## tokens
- f     -> floatdcl | can only have 5 digits after decimal point
- i     -> intdcl   | standar integer
- p     -> print    | print id/variable
- id    -> id       | variable, can only be lowercase letter from a to z
- 123   -> inum     | integer
- 12.4  -> fnum     | float
- \=    -> =        | assign
- \+    -> +        | plus
